Cricket Highlights Generator

Overview
This project is a Python-based application that generates cricket match highlights from a full-length video. It extracts important moments such as fours, sixes, and wickets using predefined timestamps and combines them into a short highlights video.

---

Objective
- Reduce long match videos into short highlights
- Extract important events using timestamps
- Demonstrate video processing using Python

---

Technologies Used
- Python
- MoviePy
- Pandas
